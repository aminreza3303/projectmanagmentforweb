MVP Scope

In Scope
- Basic auth (register/login/logout)
- Roles: team member & project manager (admin optional)
- Projects CRUD (core fields)
- Tasks CRUD + assignment
- Task status: in_progress / done / on_hold
- Simple dashboards per role
- Manual resource allocation (no auto algorithm)
- Simple notifications list
- Simple reports (project progress, task counts, cost summary)

Out of Scope (for MVP)
- Intelligent auto resource allocation
- Advanced analytics dashboards
- Real‑time chat
- Detailed audit/version history
- Third‑party integrations

MVP Success Criteria
- Manager can create and manage projects and tasks
- Member can view assigned tasks and update status
- Resources can be recorded and viewed
- Progress report can be generated
