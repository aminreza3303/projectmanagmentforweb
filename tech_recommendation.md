Tech Recommendation

Goal: A simple, maintainable stack with a clear FE/BE separation.

Back‑End
- Node.js + TypeScript
- Express
- Auth: JWT
- Validation: Zod
- API Docs: Swagger
- File uploads: Multer
- ORM: Prisma

Database
- SQLite (local development)

Front‑End
- Vue 3 + Vite
- Pinia for state
- Bootstrap 5 for UI
- Axios for API calls

Why this stack
- Prisma simplifies schema + migrations
- SQLite is easy for local demos
- Vue + Bootstrap helps build UI fast
- Clear separation between FE and BE
