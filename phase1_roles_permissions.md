Roles & Permissions

1) Team Member
- View projects they belong to
- View assigned tasks with details
- Update status of own tasks
- Update project status for projects they belong to
- View allocated resources related to their tasks
- View limited reports (only their projects)

2) Project Manager
- Create/update/delete projects
- Manage project members
- Create/update/delete tasks and assign members
- Allocate resources to projects/tasks
- View and generate reports
- View managerial notifications

3) Admin (optional)
- Manage users and roles
- View all projects and reports
- Manage global system settings

Access Rules
- Users can access only projects they are members of.
- Project managers have full access to their projects.
- Admin has access to all resources.
