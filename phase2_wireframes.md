Wireframes (Notion‑inspired)

1) Dashboard / Home
- Top bar: Search, Updates, user avatar
- Left sidebar: workspace switcher + Inbox, Favorites, Teamspaces, Shared, Private
- Main area: welcome + summary cards (projects, tasks, resources)
- Lower section: Recent projects list (Notion‑style)

2) Projects (Database view)
- Title + “New” button
- View tabs: Table / Board / Calendar
- Table columns: Title, Status, Priority, Dates, Manager
- Filters/Sort (top right)

3) Project Detail (Notion page)
- Large title + icon
- Properties panel: status, priority, dates, manager
- Content section: description + links
- Embedded databases: Tasks / Resources / Reports

4) Tasks (Database view)
- Title + “New Task”
- View tabs: Table / Board / Calendar
- Columns: Title, Assignee, Due date, Status, Priority
- Filters: status, assignee, date

5) Resources (Database view)
- Title + “New Resource”
- Columns: Type, Amount, Unit, Project/Task
- Optional summary chart below

6) Reports (Hybrid view)
- Filters at top
- Table results (Database view)
- Summary charts below

7) Notifications (Inbox)
- List with unread marker
- Filters: All / Unread
